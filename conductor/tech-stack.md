# Technology Stack

## Core Technologies
- **Language**: C# (utilizing latest language features where applicable).
- **Core Libraries Target**: .NET Standard 2.0 (for maximum compatibility across .NET platforms).
- **Apps and Tests Target**: .NET 8.0 (utilizing the latest performance and runtime features).

## Primary Dependencies
- **Serialization**: `Newtonsoft.Json` (used for message serialization and persistence).
- **Logging**: `Microsoft.Extensions.Logging` (providing a standardized logging abstraction).
- **Networking**: System.Net.Sockets (UDP Multicast).

## Development and Testing
- **Test Framework**: xUnit.
- **Build System**: .NET CLI / MSBuild.
- **Dependency Management**: NuGet.
